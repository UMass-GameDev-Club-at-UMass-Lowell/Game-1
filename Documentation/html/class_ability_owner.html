Detailed Description

Handles timed events related to abilities. Specifically, charge ups, activating the ability, repeated events during duration time (such as healing over time), and cooldowns. It also holds useful data, namely the OwnerTransform (playerâ€™s transform), the ability info this ability is using, and the current ability state (ready to use vs on cooldown).

Documentation updated 8/21/2024

Note
This script does not inherit from monobehavior, so it does not have many of the default unity functions like Start() and Update().
