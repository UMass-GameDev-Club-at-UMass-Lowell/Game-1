Detailed Description

This is the info for the fire ability set made for the club fair build. The offense ability shoots bursts of fireballs. The defense ability spawns a tall fire on each side of the player. The utility ability makes the player immune to fire for a short time. The passive ability makes the player take less fire damage. This is a scriptable object, meaning you can make an instance of it in the editor.
